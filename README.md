# NEAT
http://nn.cs.utexas.edu/downloads/papers/stanley.ec02.pdf

## Build and run:
1. Install python3
2. Install scons:
```sh
python3 -m pip install scons
```
3. Clone the project:
```
git clone https://github.com/Schlomon/NEAT.git
cd NEAT
```
4. Build with scons:
```
scons
```
5. Execute
```
cd bin/
neat.exe
```
